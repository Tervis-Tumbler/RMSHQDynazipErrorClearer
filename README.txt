Place the "RMSHQDynazipErrorClearer" folder in the following directory:
%USERPROFILE%\Documents\WindowsPowerShell\RMSHQDynazipErrorClearer

Run the following in PowerShell as an Administrator:
New-RMSHQDynazipErrorClearerScheduledTask

This will create a scheduled task that will run the EXE on next startup.
